## 阶段间依赖关系

```
P0 (骨架+采集) ──→ P1 (AI加工) ──→ P2 (草稿+后台) ──→ P3 (发布+部署) ──→ P4 (联调)
     │                                    │
     └── 可提前做 US-005 Docker ──────────┘
```

P0~P4 严格串行，但阶段内有大量并行机会。P2 是并行空间最大的阶段（后端草稿 + 前端骨架可同时开发）。
